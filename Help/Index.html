About NoNoun

NoNoun is a Chrome Extension that allows you to replace gender-specific pronouns (“he”, “she”), objects (“his”, “her”), and possessives (“his”, “her/hers”) with gender-neutral alternatives on English language Web pages.

How to Install

Install from the Chrome Web Store.

How to Use

First

  • Make sure you are using the Chrome browser. At this time, NoNoun does not work in other browsers.
  • Install the NoNoun Chrome extension (see instructions above).
  • Go to a Web Page you want to make gender neutral.

Click the NoNoun icon in the Chrome menu bar.

Choose what substitutions you would like to make (or leave as is to accept defaults). Then click "Go".

The Her/Her Problem

NoNoun uses simple pattern matches to make replacements. This can lead to mistakes with the word “her”, which is sometimes an object (“I talked to her.”) and sometimes a possessive (“I talked to her mother.”).

Questions & Answers

Can I control what gender-neutral words to use?

Yes. By default, NoNoun uses “o” for “he” and “she”, “om” for “him” and “her”, and “oz” for “his” and “hers”. But you can specify any words you like.

Can I control which gender-specific words to replace?

Yes. For example, you may decide that because “she”, “her”, and “hers” are rarely used in gender-neutral ways, you want to replace “he”, “him”, and “his”, but leave “she”, “her”, and “hers” alone. To do this, leave the replacement boxes for the words to leave alone empty. Or, fill the replacement boxes with the same value (replace “her” with “her”).

Disclaimers

  • NoNoun uses simple pattern matching for replacements. It may make mistakes. It may miss some locations or accidentally act on some locations it shouldn't. See the her/her problem as one other potential issue.
  • NoNoun may not work on misbehaving Web pages (pages that generate errors that are often not seen by the reader).
